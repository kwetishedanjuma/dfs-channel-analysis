# Statistical Analysis

Main Python scripts for analysing channel geometry, environmental distributions, and downstream patterns in DFS.

## Analysis Components
- Channel width trend analysis (OLS regression)
- Environmental proportion calculations
- Multi-scale statistical comparisons
- Cross-system pattern identification

## Key Scripts
- Data processing and cleaning functions
- Statistical analysis and regression modelling
- Visualization and plotting utilities
- Results export and reporting tools

## Dependencies
- Python 3.x
- pandas, numpy, matplotlib, scipy
- scienceplots for publication-quality figures

## Workflow
Processes standardised datasets from all five systems using unified methodology to enable robust cross-system comparisons.
